Tentang EkonoMap

Portal visualisasi data ekonomi Indonesia

🎯 Tujuan

EkonoMap adalah portal sumber informasi pertumbuhan ekonomi Indonesia. Kami menyajikan data indikator makroekonomi — Produk Domestik Bruto (PDB), inflasi, serta nilai ekspor-impor — dalam bentuk visualisasi interaktif yang mudah dipahami oleh pelajar, peneliti, maupun masyarakat umum.

📊 Sumber Data
  • 🏛️ Badan Pusat Statistik (BPS) — webapi.bps.go.id
  • 🏦 Bank Indonesia — data inflasi & moneter
  • 🌐 World Bank — data historis global
⚙️ Teknologi
  • 💻 HTML5, CSS3, JavaScript (Vanilla)
  • 📈 Chart.js untuk visualisasi
  • 🗄️ Backend & database oleh tim data
  • 🚀 Deployment di ekonomap.site
Layanan ini menggunakan API Badan Pusat Statistik (BPS)
